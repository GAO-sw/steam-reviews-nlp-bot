# Steam Reviews Analysis Telegram Bot

**Тема курсовой работы:** Разработка Telegram-бота для визуализации и анализа отзывов об играх в Steam с использованием NLP-моделей.
**Студент:** Сунвань Гао

## Описание проекта (Project Description)
Инструмент для анализа и визуализации отзывов в Steam. 
Программа скачивает отзывы по ID игры с помощью официального API (`steamreviews`), анализирует тональность текста с использованием NLP-моделей и визуализирует данные (Pandas, Matplotlib). Взаимодействие с пользователем реализовано через Telegram-бота.

## Стек технологий (Tech Stack)
- Python
- Pandas, Matplotlib
- NLP библиотеки
- Telegram Bot API
- `steamreviews` API
