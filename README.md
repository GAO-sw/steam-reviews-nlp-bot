## 📝 Описание проекта 
Данный проект представляет собой минимально жизнеспособный продукт  Telegram-бота, предназначенного для автоматического парсинга, тонального анализа и визуализации отзывов на игры в платформе Steam.

### Основной функционал:
1. **Интерактивность:** Пользователь отправляет числовой `AppID` игры (например, `730` для CS2).
2. **Сбор данных:** Бот обращается к официальному Steam API и выгружает последние 100 англоязычных отзывов.
3. **NLP Анализ:** Текст отзывов обрабатывается библиотекой `pandas` и NLP-инструментом `TextBlob`. Вычисляется тональность (Positive/Negative/Neutral) и частота упоминания ключевых слов (как позитивных, так и негативных).
4. **Визуализация:** На основе анализа динамически генерируются два графика (круговая диаграмма распределения тональности и столбчатая диаграмма частоты слов) без промежуточного сохранения на сервере (через буфер памяти `io.BytesIO`).
